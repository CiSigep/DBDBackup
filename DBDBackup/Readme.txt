This application is intended for backing up Dead By Daylight saves.

WARNING: This application is not confirmed to not trigger an EasyAntiCheat ban. Use at your own risk.

Installation is as follows:
1. Compile the application and place it and BackupConfig.xml where you want
2. Change the values in BackupConfig.xml to your preferred settings

This application will launch Dead By Daylight and while the game is running, create backups based on the config. It can be configured to save a backup only on game exit, periodically while the game is running, or watch the game save directory for changes and copy them.
